# Mobile

## PAra criar o projeto
1. cd\ -- para entrar na pasta raiz do ocmuputador
2. md (nome) -- cria a pasta em que o projeto irá ser hospedado
3. cd (pasta criada) -- muda para a pasta criada
4. npx @react-native-community/cli init (nome do projeto)

## Para rodar o projeto
1. code . -- no terminal anterior entre no vs code
2. cd (projeto) -- para entrar no projeto
3. npx react-native run-android -- para rodar o projeto
